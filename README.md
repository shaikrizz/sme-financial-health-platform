\# SME Financial Health Assessment Platform



Streamlit-based financial health assessment tool for SMEs.



\## Features

\- Financial overview (Revenue, Expense, Net)

\- Risk \& creditworthiness indicator

\- AI insights (local AI + fallback)

\- Expense breakdown + monthly charts

\- Financial forecasting

\- Industry benchmarking

\- Investor-ready report export (PDF/TXT)

\- Security \& compliance notes (privacy-first demo)



\## Input Format

CSV/XLSX columns: Date, Category, Amount, Description  

Category values: Revenue / Expense (case-insensitive)



\## Run Locally

python -m pip install -r requirements.txt

python -m streamlit run app.py
